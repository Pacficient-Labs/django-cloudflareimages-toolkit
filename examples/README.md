# Examples

Standalone scripts that demonstrate specific package features. These are
not part of the installed wheel — they live alongside the package source
for reference and copy-paste use.

| File | Demonstrates |
|---|---|
| `cloudflareimagefield.py` | Using `CloudflareImageField` on a Django model, with form integration and admin display. |

Run any example as a Django script after `python manage.py shell -c "exec(open('examples/cloudflareimagefield.py').read())"`, or copy fragments into your own models.
