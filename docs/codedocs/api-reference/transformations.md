## `CloudflareImageTransform`

```python
class CloudflareImageTransform:
    def __init__(self, base_url: str, zone: str | None = None) -> None: ...
```

### Constructor options

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `base_url` | `str` | — | Base delivery URL or image path. |
| `zone` | `str \| None` | `None` | Explicit domain for `/cdn-cgi/image/` output. |

Example:

```python
url = (
    CloudflareImageTransform(base_url)
    .width(800)
    .height(450)
    .fit("cover")
    .quality(80)
    .format("auto")
    .build()
)
```

## `CloudflareImageUtils`

All methods are `@staticmethod`.

```python
def extract_image_id(url: str) -> str | None: ...
def is_cloudflare_image_url(url: str) -> bool: ...
def get_srcset(base_url: str, widths: list[int], quality: int = 85) -> str: ...
def get_sizes_attribute(breakpoints: dict[str, int]) -> str: ...
def validate_image_url(url: str) -> bool: ...
```

Combined example:

```python
base_url = "https://imagedelivery.net/account-hash/demo-image/public"

thumbnail = CloudflareImageVariants.thumbnail(base_url, 200)
srcset = CloudflareImageUtils.get_srcset(base_url, [320, 640, 960] quality=80)
is_valid = CloudflareImageUtils.validate_image_url(base_url)
```
